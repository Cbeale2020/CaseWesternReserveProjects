The ER diagram was created using two datasets from Kaggle:
1. Movies on Streaming Networks
2. Movie Metadata Set

The two datasets were merged utilizing the "Title" and "Year". The merged data was cleaned to include only the columns needed for the movie query database.

The movies_table was created with a unique movie id for each movie. Each second normal form table is connected to the movie_table via the unique movie ID.
