Movie Analysis by Ratings

by Ben Stork, Crisharon Beale, Robert Payne, Julie Pyle

Our team created a movie information data frame in SQL (postgres) that allows users to quickly and conveniently search for the top movies by ratings. The data was obtained from two datasets on Kaggle. The data was cleaned and put into second normal form for analysis. The following rating systems were utilized:
1. Rotten Tomatoes
2. IMDB

We analyzed the database for the Top 10 movies for each rating system by the following genres:
1. Action
2. Comedy
3. Documentary
4. Drama
5. Family
6. Horror
7. Sci-fi
8. Western
